# Examples

This directory contains small PHCL-TERRAFORM examples you can inspect and run.

For broader PHCL documentation and additional examples, see the main PHCL repository:

- https://github.com/nexusproject/phcl

## Included

- `aws-basic/`
  - A small AWS resource example with a provider, a resource, nested blocks, and an output.
- `tfvars-functions/`
  - A minimal example showing `encode_tfvars`, `decode_tfvars`, and `encode_expr`.

Each example folder contains same-named source and generated files, plus a `README.md`.
